Image Storage Strategy

Original uploads and AI results are stored in S3. The API generates signed URLs valid for 1 hour to prevent public hotlinking.

s3://vto-mvp/uploads/{user_id}/{photo_id}.jpg

Horizontal Scalability

The REST API is stateless. It uses JWT for authentication, allowing any instance of the server to handle any request.
